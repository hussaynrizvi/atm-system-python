# ATM System (Python)

## Description
A simple ATM system built using Python that allows users to perform basic banking operations.

## Features
- Create account
- Deposit money
- Withdraw money
- Check balance
- Data stored using JSON

## Technologies Used
- Python
- JSON

## Files
- atm.py → main program
- accounts.json → stores user data

## How to Run
1. Download the files
2. Run atm.py
3. Follow the instructions
